Radar Signal Processing Chain — Simulator

A working, in-browser model of a pulsed radar's transmit and receive signal chain: baseband waveform generation → digital/analog up-conversion → RF → the antenna, and back down through RF → analog/digital down-conversion → pulse compression → range recovery. Move the sliders and watch the spectra, waveform, and range estimate respond in real time.

Block functions

  • Waveform generator — create baseband I/Q pulse (LFM chirp)
  • DUC — interpolate, mix with NCO, filter → digital IF
  • DAC / Up converter — digital IF → analog IF → RF
  • T/R switch — shares the antenna between transmit and receive
  • Down converter / ADC — RF → analog IF → digitized IF
  • DDC — mix with NCO, filter, decimate → baseband I/Q
  • Digital signal processing — matched filter (pulse compression) → range

Key relationships

Transmit: fRF = fLO + fIF

Receive: fIF = | fRF − fLO |

Range resolution: ΔR = c / (2B)

Unambiguous range: Ru = c / (2·PRF)

Output of DUC / DDC is complex baseband I/Q.

Notes

  • One T/R module handles transmit and receive.
  • Tx path: baseband → RF (up-conversion) → antenna.
  • Rx path: antenna → RF → baseband (down-conversion) → matched filter.
  • The analog IF / RF spectrum cards are illustrative shapes drawn to scale; the baseband and digital-IF cards are real FFTs of the simulated samples.
  • The K / M pills show the interpolation/decimation factor really required for your chosen fIF, B, and fs; the DUC/DDC animation itself always runs at a fixed illustrative factor so it stays smooth in the browser.
  • Frequencies, sample rates, and the range estimate are for education, not a specific system.
